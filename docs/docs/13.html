    These set of functions exist for the purpose of working with users, or because they are user-related functions. You can add new ones, drop old ones, or edit them. Click a function listed below to get a more detailed description of how they work

  • connect - Connects to the txtSQL Service
  • disconnect - Disconnects from the txtSQL Service
  • grant_permissions - Allows for the creation, deletion, and editing of a user
  • getUsers - Returns a list containing all of the txtSQL users
  • version - Returns the current txtSQL version
